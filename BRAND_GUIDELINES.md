# راهنمای برند IOP

این سند راهنمای استفاده از برند Iran Open Privacy (IOP) را ارائه می‌دهد.

## لوگو

لوگوی IOP شامل یک آیکون سپر با قفل در داخل و متن "IOP" و "Iran Open Privacy" است.

### عناصر لوگو

- **آیکون سپر**: نماد حفاظت و امنیت
- **قفل**: نماد حریم خصوصی و امنیت داده‌ها
- **متن IOP**: حروف بزرگ و بولد
- **متن Iran Open Privacy**: نام کامل پروژه

### استفاده از لوگو

لوگو در سه اندازه در دسترس است:
- **Small**: برای استفاده در فوتر و فضاهای کوچک
- **Medium**: برای استفاده در هدر و ناوبری
- **Large**: برای استفاده در صفحات اصلی و هیرو سکشن‌ها

## رنگ‌ها

### رنگ اصلی
- **سیاه**: `#000000` - برای متن و عناصر اصلی
- **سفید**: `#FFFFFF` - برای پس‌زمینه
- **آبی**: `#276EF6` - رنگ آکسان برای دکمه‌ها و لینک‌ها
- **خاکستری**: `#666666` - برای متن‌های ثانویه

### استفاده از رنگ‌ها

- از رنگ آبی فقط برای عناصر تعاملی (دکمه‌ها، لینک‌ها) استفاده کنید
- از گرادیان، سایه و افکت استفاده نکنید
- طراحی باید مینیمال و تمیز باشد

## تایپوگرافی

### فونت اصلی
- **Vazirmatn**: فونت اصلی برای متن فارسی
- **IRANSans**: فونت جایگزین
- **System UI**: فونت fallback

### سایزهای فونت

- **عنوان اصلی**: 5xl (48px) - برای هیرو
- **عنوان بخش**: 2xl-3xl (24-30px) - برای بخش‌ها
- **متن بدن**: lg (18px) - برای محتوای اصلی
- **متن کوچک**: sm (14px) - برای اطلاعات فرعی

## اصول طراحی

### مینیمالیسم
- طراحی باید ساده و تمیز باشد
- از عناصر غیرضروری اجتناب کنید
- فضای سفید کافی برای خوانایی

### حرفه‌ای بودن
- طراحی باید شبیه به سایت‌های foundation مانند W3C، Mozilla، OWASP باشد
- استفاده از خطوط مستقیم و بلوک‌های تمیز
- بدون تزئینات اضافی

### RTL Support
- تمام طراحی باید از راست به چپ باشد
- فونت‌های فارسی با پشتیبانی کامل RTL

## فایل‌های لوگو

- `/public/logo-icon.svg` - آیکون لوگو (برای favicon)
- `/public/logo-full.svg` - لوگوی کامل با متن
- `/components/Logo.tsx` - کامپوننت React برای استفاده در صفحات

## استفاده در کد

```tsx
import Logo from '@/components/Logo'

// استفاده ساده
<Logo />

// با اندازه مشخص
<Logo size="large" />

// بدون متن
<Logo showText={false} />
```

## محدودیت‌ها

- لوگو را تغییر رنگ ندهید (فقط سیاه و سفید)
- نسبت ابعاد لوگو را حفظ کنید
- از لوگو در پس‌زمینه رنگی استفاده نکنید (فقط سفید)
- لوگو را کوچک‌تر از حداقل اندازه (24px) نکنید
